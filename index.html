Services.

Développement Front-End

Création d'interfaces web attrayant et fonctionnel avec des interactions dynamiques grâce à des animations CSS et des bibliothèques telles que React pour offrir une expérience utilisateur fluide.

Développement Front-End

  • Intégration d'éléments sur une page web conformément à une maquette Figma avec HTML et CSS.
  • Mise en œuvre d'animations CSS.
  • Développement d'interface web avec SASS.
  • Développement d'éléments grâce à des composants React.
  • Configuration de la navigation entre différentes pages de l'application React avec React Router.
  • Optimisation des performances d'un site web.
  • Déploiement en ligne d'un site.

Programmation et Gestion de Projet

Installation et configuration d'environnement de développement, versionnage de code avec Git et GitHub. Utilisation d'outils de gestion de projet Kanban et outil de veille.

Programmation et Gestion de Projet

  • Versionnage d'un projet avec Git et GitHub.
  • Programmation en utilisant les fondamentaux de JavaScript.
  • Manipulation des éléments du DOM avec JavaScript
  • Surveillance du déroulement du projet grâce à un outil de gestion de projet (Tableau Kanban).
  • Connaissance de la méthode Agile.

Déploiement et Maintenance

Déboguer de problèmes à l'aide d'outils de développement. Rédaction des scénarios de test pour assurer la qualité du site et de mise en place des interactions avec le back-end via des appels API et des configurations de route.

Déploiement et Maintenance

  • Débogage d'un site web grâce aux Chrome DevTools.
  • Configuration des routes API pour la communication client / serveur.
  • Implémentation de la gestion des données avec Redux pour assurer le fonctionnement du front.
  • Affichage des données du back end sur l'interface via des appels API.
