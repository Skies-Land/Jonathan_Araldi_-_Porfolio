FAQ.

Les outils que j'utilise...

💻 - Pour coder mes projets, j'utilise VS Code sous les systèmes d'exploitation Windows/macOS.

🌐 - Pour développer des sites web, j'utilise HTML, CSS, JavaScript, React et SASS pour créer des interfaces utilisateur interactives et attrayantes.

👾 - Pour le versionnage de code, j'utilise Git et GitHub pour stocker mes projets en ligne.

🚀 - Pour le déploiement, j'utilise GitHub Pages pour héberger mes projets en ligne.

💬 - Pour la communication, j'utilise Slack et Discord pour échanger avec d'autres développeurs.

⚙️ - Pour gérer les performances de mes sites web, j'utilise Lighthouse pour auditer et optimiser les performances de mes sites web combiner avec l'extension WAVE pour améliorer l'accessibilité.

🧠 - Pour continuer d'apprendre, j'utilise les ressources OpenClassrooms / Udemy / Stack Overflow / YouTube / les documentation des librairie.

Ce qui m'attire dans le développement web...

J'ai toujours été fasciné par la façon dont les sites web peuvent impacter et faciliter la vie quotidienne des utilisateurs.

Le développement web offre une combinaison unique de créativité et de logique, où je peux constamment apprendre et relever de nouveaux défis techniques.

De plus, être capable de créer quelque chose de tangible et accessible à tous sur internet est extrêmement gratifiant pour moi.

Mes objectifs professionnels à court, moyen et long terme sont...

À court terme, je vise à acquérir une expérience pratique en travaillant sur des projets réels, afin de renforcer mes compétences techniques et de me familiariser avec les bonnes pratiques de l'industrie.

À moyen terme, je souhaite me spécialiser davantage dans le développement front-end en explorant des technologies émergentes telles que les frameworks JavaScript avancés.

À long terme, j'aspire à élargir mes compétences en me lançant dans le développement back-end, tout en contribuant activement à des projets innovants au sein d'une équipe dynamique.

Mes qualités principales...

Je suis rigoureux dans mon approche du travail, ce qui se traduit par une attention aux détails et un souci constant de la qualité.
Je prendre le temps de bien soigner la présentation et l'indentation de mon code, puis d'ajouter des commentaires afin de faciliter le partage et la collaboration harmonieuse.

Je suis également passionné par l'apprentissage continu et je suis toujours à la recherche de nouvelles opportunités pour développer mes compétences et élargir mes horizons professionnels.
