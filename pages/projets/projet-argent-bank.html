Projet Argent Bank.

  • Implémentez le front-end d'une application bancaire avec React et Redux pour créer une expérience utilisateur dynamique et réactive.

  • Dans ce projet, j'ai intégré Argent Bank en tant que développeur front-end. Argent Bank est une nouvelle banque en ligne qui souhaite percer dans le secteur bancaire.

    Mon rôle a été de participer au développement de la partie d'authentification des utilisateurs de l'application. L'objectif était de créer une application,

    - web complet et responsive avec React.
    - avec utilisation de Redux pour gérer le state de l'ensemble de l'application.
    - avec gestion d'appel API avec le serveur pour la récupération des données utilisateur.
    - et dotée de fonctionnalités pour l'utilisateur comme : connexion, déconnexion de session. Modification du nom d'utilisateur via un formulaire et consultation de ses données.

  • - Afficher les données du back end sur l'interface via des appels API


    - Configurer des routes API pour la communication client / serveur


    - Implémenter la gestion des données avec Redux pour assurer le fonctionnement du front

  • Pour tester le projet, il vous faut récupérer mon repository et exécuter les commandes suivante :

    Pour installer les dépendances de ce projet
    => npm install

    Pour lancer la base de données en local
    => npm run dev:server

    Pour lancer le serveur de développement dans un autre terminal à partir du dossier front-end
    => npm run dev

Mon retour

Jonathan

Dans ce projet, j'ai travaillé sur le développement front-end d'une application bancaire en utilisant React et Redux. Mon principal objectif était d'intégrer le front-end avec le back-end via des appels API, pour assurer une communication fluide entre le client et le serveur.

Pour commencer, j'ai installé le back-end de l'application et testé la route de connexion pour m'assurer de son bon fonctionnement. Ensuite, j'ai créé l'application React que j'ai initialisé avec Vite.js, intégré les différentes pages HTML/CSS de la maquette avec React et géré les routes avec React Router.

J'ai mis en place un state global avec Redux pour gérer les données de l'application, notamment la connexion utilisateur. J'ai assuré que la gestion de la connexion utilisateur était fonctionnelle avec le store de Redux. Par la suite, j'ai sécurisé la connexion de l'utilisateur et implémenté une fonctionnalité clé de l'application : la modification du nom d'utilisateur via un formulaire. J'ai veillé à ce que les modifications persistent et soient cohérentes sur l'ensemble de la page.

Grâce à ce projet, j'ai renforcé mes connaisances avec React et fait une première utilisation avec Redux. Cette expérience m'a permis de comprendre l'importance d'une communication fluide entre le front-end et le back-end, tout en maintenant un état global cohérent à travers l'application.
