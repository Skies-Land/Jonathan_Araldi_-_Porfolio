Projet 724events.

  • Débuggez le site d'une agence d'événementiel avec utilisation de Chrome DevTools pour identifier et comprendre les problèmes et bugs signalés dans le site web actuel.

  • L'agence 724events souhaite publier la nouvelle version de son site vitrine. Le design a été validé, et un premier développeur freelance a commencé l’intégration. Malheureusement, ce freelance a dû abandonner le projet pour raisons personnelles. Le site est fonctionnel, mais quelques bus entravent le bon usage par les visiteurs.

    Problème à corriger :

    - Le slider : Son but et de mettre en avant quelques-uns de leur réalisation, mais une image blanche apparaît. L'ordre de défilement des images n'est pas le bon. Les images doivent s'afficher de plus ancien au plus récente.

    - Section nos réalisations : Lorsque que l'utilisateur choisi une catégorie, il ne se passe rien. Il faut que le filtre s'applique et affiche que les éléments de la catégorie sélectionnée. Certaines cartes n'affichent pas les bonnes informations.

    - Formulaire de contact : Lorsque l'utilisateur envoie un message, il ne se passe rien. Un message de confirmation d'envoi doit s'afficher.

    - Vignette dans le footer : Permet d'afficher une dernière prestation qui se compose par une image, d'un nom et du mois. Cette élément doit être finalisé.

Mon retour

Jonathan

Pour ce projet, j'ai commencé par examiner le code existant avec les Chrome DevTools pour identifier et comprendre les bugs signalés. Ensuite, j'ai utilisé mes compétences en JavaScript et les outils React, comme React Developer Tools, pour résoudre ces problèmes efficacement.

J'ai également complété un cahier de recette détaillant tous les scénarios de test nécessaires pour vérifier le bon fonctionnement du site, incluant des tests unitaires et fonctionnels. Pour gérer le projet, j'ai utilisé Yarn comme gestionnaire de packages et Node.js pour exécuter du code JavaScript côté serveur, ce qui m'a aidé à gérer les dépendances et à exécuter des tests localement.

Tout au long du projet, j'ai versionné mon travail avec GitHub et utilisé Visual Studio Code pour développer et déboguer le site. Ce projet m'a permis de développer mes compétences en débogage et d'approfondir ma maîtrise des outils comme React Developer Tools, Yarn et Node.js. Grâce à ce projet, j'ai pu garantir la qualité et la fiabilité du site web, tout en me préparant à travailler sur des projets web complexe.
