Projet Sophie Bluel.

  • Créer une page web dynamique pour le site portfolio d'une architecte d'intérieur en utilisant JavaScript et en communiquant avec une API.

  • Dans l'agence ArchiWebos, j'ai été envoyé en renfort pour participer à la conception d'un site portfolio d'une architecte d'intérieur.

    Ma mission a été de développer :

    - La page de présentation des travaux de l’architecte

    - La page de connexion de l’administrateur du site

    - Une modale permettant de supprimer et l’upload de médias


    J'ai utilisé JavaScript pour gérer les interactions des utilisateurs et communiquer avec une API Swagger pour récupérer et envoyer des données.

  • - Gérer les événements utilisateurs avec JavaScript


    - Manipuler les éléments du DOM avec JavaScript


    - Récupérer les données utilisateurs dans le JavaScript via des formulaires

  • Pour tester ce projet, il vous faut récupérer le backend sur mon repository. Les identifiants nécessaires pour accéder à la page d'administration sont les suivants :


    - Email: sophie.bluel@test.tld

    - Password: S0phie

Mon retour

Jonathan

Avoir la capacité de concevoir des pages web dynamique avec JavaScript est fondamental dans le monde du développement web.
