Projet Kasa.

  • Créez une application web de location immobilière avec React

  • Kasa, une entreprise de location d’appartements entre particuliers en France. La CTO, a lancé une refonte totale pour passer à une stack complète en JavaScript avec Node.js côté back-end, et React côté front-end.

    Ma mission a été de développer l’ensemble de l’application avec React, de configurer les routes avec React-router et pour initialiser l'application, j'ai utilisé Vite.js.

  • - Configurer la navigation entre les pages de l’application avec React-router


    - Développer des éléments de l’interface d’un site web grâce à des composants React


    - Mettre en œuvre des animation CSS


    - Développer une interface avec SASS


    - Initialiser une application avec Vite.js

Mon retour

Jonathan

Prise de décision (recommandation de mon mentor) : J’ai configuré une méthode permettant de récupérer les données d’un fichier JSON avec fetch pour anticiper l’ajout d’une API distant pour le back-end.

En réalisant ce projet, j'ai pu me familiariser avec React et React-router, combiner avec l'utilisation de SASS pour concevoir des applications web dynamique et réactif.

J'ai organisé les composants React et le style SCSS par dossier qui les caractérise en suivant la logique d'un design pattern similaire au pattern 7.1 en SASS.
Cela rend beaucoup plus simples la recherche et la gestion de fichiers pour modifier un élément du projet.
